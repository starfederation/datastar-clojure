# Malli schemas for the SDK

## Installation

Install using clojars deps coordinates:

[![Clojars Project](https://img.shields.io/clojars/v/dev.data-star.clojure/aleph-malli-schemas.svg)](https://clojars.org/dev.data-star.clojure/aleph-malli-schemas)

## Overview

This library provides Malli schemas for the Aleph adapter APIs.

Require the namespaces for which you want schema and/or instrumentation. Then
use malli's instrumentation facilities.

Notable schema namespaces:

- `starfederation.datastar.clojure.adapter.aleph-schemas`
